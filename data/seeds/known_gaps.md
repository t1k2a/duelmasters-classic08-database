# Known Gaps in Seed Data

## DMC/DMS カード独自ページ問題 (再調査で訂正済み)

> ⚠️ **訂正 (2026-06)**: 旧記述「DMC専用カードは一律取得不可」は **DMC-09 を試した結果からの過度な一般化で誤り**だった。
> 実際は**セット単位で可否が分かれる**。新しめ（2007〜2008）の一部セットは公式DBに `dmcXX-NNN` 形式の
> カード詳細ページが存在し、既存スクレイパー (run-dmc.ts / build-json.ts) で取得可能。

実地調査の結果:
1. 古いDMC/DMS（DMC-01〜33・DMS系, 2002〜2006頃）→ 公式DBに `dmcXX-NNN` 詳細ページ**なし**（例: dmc09 は全て該当なし）。取得不可。
2. リスティングページ (`?product=dmc09` 等) → Playwright でも AJAX が発火せず、一覧取得**不可**。
3. 新しめのDMC（下記「取得済み」参照, 2007〜2008）→ `dmcXX-NNN` 詳細ページ**あり**。取得可能。
4. ID連番は**大きく歯抜け**のことがある（再録カードは元のDM拡張パックIDで管理されるため）。
   例: DMC-46 は 001-005 の後 006-009 が欠番→010-012,018,020... と飛ぶ。
   `run-dmc.ts` 既定の「5連続ミスで打ち切り」では取りこぼすため、`scripts/fetch-dmc-html.ts` の
   `MAX_MISS` / `MAX_ID` を広げて回収する必要がある。

### 取得済みDMCセット (2026-06 取り込み, 計+37枚)
公式DBに ID が存在し取り込み済み: **DMC-34, DMC-37, DMC-38, DMC-40, DMC-41, DMC-42, DMC-44, DMC-46**。
- これにより極神編トライ・ゴッド・リンク《龍神ヘヴィ》《破壊神デス》《龍神メタル》(初出 DMC-40, 2008-01) ほかを収録。
- 副次的に、欠落していた DM-10《撃墜団長メッツァー》等の再録由来カードも補完された。
- 取り込み経路: `SETS=DMC-XX MAX_MISS=40 MAX_ID=150 tsx scripts/fetch-dmc-html.ts` → `tsx scripts/build-json.ts`。
- ⚠️ この取り込みは **build-json 直接経路** (data/raw → public/cards.json) で実施。
  **Postgres(prisma)側DBは未更新**（API用途。DB再構築時は別途スクレイプ/seedが必要）。

### 取得不可（公式DBに ID 無し → 二次ソース要）

- これらの専用新規カードは dmwiki / pixiv 百科等からの手動・半自動インポートが必要（未対応）。

## ボルメテウス・ホワイト・ドラゴン (調査済み)
ブリーフに「DM-06《ボルメテウス・ホワイト・ドラゴン》」と記載されていたが、
DM-01〜DM-30 全30セット（1978枚）のスクレイピング完了後も DB 内に存在しない。
**確認済み**: 「ボル」で始まる全カードを検索したが、ボルメテウスの名称は一件も存在しない。
**推定**: ボルメテウス・ホワイト・ドラゴンは DMC デッキ製品（DMC-09 勝舞神龍デッキ等）の
独自カードであり、DM 拡張パックには収録されていない可能性が高い。
**対応方針**: DMC カード独自ページ問題と同様、現行スクレイパーでは取得不可。
