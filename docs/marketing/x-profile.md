## 5. 場所(Location)欄・プロフィール補足の小ネタ（任意）

- **Location欄**: 実在地名を入れる必要はなし。フレーバーとして `2008年環境` や `殿堂リスト準拠` を入れる案あり。
  - **判断: 入れてもよい（優先度低）**。`2008年環境` は「いつのフォーマットか」の即時補足になり無害。ふざけ過ぎない範囲で。空欄でも支障なし。

## 6. アイコン・ヘッダー画像の方向性（方針のみ・画像生成はしない）

> 権利配慮の大原則: **公式のカードイラスト・ロゴをアイコン/ヘッダーに転用しない**。DB由来の自作要素・スクショに限定する。

- **アイコン（正方形・400×400目安）**:
  - 案1: `08` を主役にしたタイポグラフィ的マーク（虫めがね🔍やDBを想起させる記号を添える）。シンプルで小サイズでも判読可。
  - 案2: サイトのファビコン/ロゴがあればそれを流用（`images/top.png` のヘッダーロゴ部分をトリミング）。→ サイトとの一貫性が出るので**第一候補**。ロゴの有無は @dev に確認。
  - 避ける: カード画像そのもの、公式ロゴの切り抜き。
- **ヘッダー（1500×500目安）**:
  - サイトのUIスクショを帯状にレイアウト（`images/search-ui.png`＝検索UI、`images/meta.png`＝Tier表）。「検索できる／組める／Tierが分かる」が一目で伝わる。
  - ヘッダー内に小さく `非公式ファンメイド / 非営利` と `#デュエマクラシック08` を焼き込むと、初見でも立ち位置が伝わる。
- **用意すべき成果物**: 400×400 アイコン1枚 / 1500×500 ヘッダー1枚。既存 `docs/marketing/images/`（top/search-ui/meta/card-detail）を素材に流用可能。実制作が必要なら @dev にロゴ有無と書き出しを相談。
