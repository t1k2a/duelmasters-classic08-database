# デスクトップアプリ用｜X投稿リマインダー・ルーティン

Claude デスクトップアプリ（または claude.ai）の「スケジュール／ルーティン」で
**1日1回・毎日 19:33** に実行する設定を1つ作り、下の「貼り付けプロンプト」をそのまま貼るだけ。
その日に投稿すべき文面を毎日提示してくれる（休養日・振り返り日はその旨を伝える）。
カレンダーは一切使わない。実際の投稿はコピペで手動。

## 設定手順（1回だけ）
1. デスクトップアプリ／claude.ai で新しいスケジュール（ルーティン／定期タスク）を作成
2. 実行間隔: **毎日 19:33**（各投稿時刻より前に届く。earliest 20:00 の枠にも間に合う）
3. 下の「貼り付けプロンプト」を本文に貼って保存
4. 通知は Claude 側でOK

---

## 貼り付けプロンプト（ここから下をコピー）

```
あなたは私のX(Twitter)投稿リマインダーです。今日の日付を確認し、下の「投稿スケジュール」から今日に該当する項目を探して、次の形式で簡潔に通知してください。

- 今日が投稿日: 「本日 HH:MM に投稿」＋その日の投稿文面を、コピペしやすいようそのまま提示（複数投稿=スレッドなら順番付きで全部）。1日1〜2本まで。
- 今日が休養日／振り返り日: 「本日は投稿なし（休養）」または「本日は振り返り: <やること>」だけ短く伝える。
- どの日付にも該当しない（スケジュール期間外）: 「本日の予定はありません」とだけ伝える。

余計な前置きは不要。ハッシュタグは #デュエマクラシック08 に統一。URLは本文のまま残す。
```
